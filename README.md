# Chat Commands
- #bi
- #bi help
- #bi copy
- #bi show

## `#bi`
Display UID in chat and copy to clipboard.

## `#bi help`
Display all commands in chat.

## `#bi show`
Display UID in chat.

## `#bi copy`
Copy to your clipboard. Only working for PC/Desktop.
